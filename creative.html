This polygon representation of a Poppy flower was created using an original photograph that was traced with the PEN TOOL to create a myriad of connected triangles to fill the shape of the petals. This artwork was time consuming beacuse of the small triangles sizes, reuired to achieve the detail. The fill of the poly triangle was set by using the EYE-DROP TOOL to pull colours from the original image.

This isometric design was created by creating a square grid and rotating it 45 degrees, then squashing it to 52% of it's orginial height. I then used this as the basis for all lines that were included in the creation of the walls, floors and stairs. I then used a colour palette of just 2 colours with each of them a secondary lighter version. I then assigned each colour to a direction within the iso design and filled it in.

Ocean Cut-Out

Illustrator

  • #0f7EC1
  • #1EB3E4
  • #74C7EA
  • #D2EBE4
  • #EEF1F3

To create theis paper cut-out effect I began by drawing a random blob shape and cutting it out from a shape layer. I then copied this and pasted it above and expanded it, then repeated this step multiple times. To colour the layers I created a gradient and used the EYE_DROP TOOL to fill each level of the design. Finally I created sea-life shape layers and applied a large, blurred drop shadow to give the 3D effect.
